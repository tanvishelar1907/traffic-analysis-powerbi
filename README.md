# Traffic Data Analysis Dashboard (Power BI)

## Project Overview
This project analyzes traffic data using Power BI to identify traffic patterns, peak congestion hours, and trends through interactive visualizations.

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization

## Key Insights
- Peak traffic hours during morning and evening.
- Weekdays show higher traffic volume compared to weekends.
- Certain locations experience higher congestion.
